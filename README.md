# -RK3588-
2026嵌赛芯片应用赛道瑞芯微赛题参赛作品，本项目专为听障人群打造智能AR户外辅助头盔。系统以基于RK3588的ELF2边缘节点为算力核心，通过低功耗蓝牙与专属手机App无缝交互，实时获取定制导航流。视觉渲染上，系统深度调用底层Mali-G610 GPU硬件加速与Vulkan图形接口，配合NanoVG引擎在双目AR模组上高帧率绘制全景HUD路书。感知交互层则全面释放内置6 TOPS NPU算力，本地部署AI大模型实现环境声音方位识别与视觉预警；同时集成红外摄像与红外补光模组，达成复杂光线下的高精手部追踪与实时手语翻译。项目将跨维度信息彻底“视觉化”，重塑出行安全与沟通独立性。
